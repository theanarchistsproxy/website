Welcome to The AP or The Anarchist's Proxy

About

The AP is a pure anarchy Minecraft server started on December 25th 2025. As a pure anarchy Minecraft server you can do whatever you want: hacking, griefing, killing, etc.

How to Connect

  • First, open Minecraft.
  • Secondly, go to the multiplayer menu.
  • Thirdly, click add server.
  • Finally, write "The Anarchist's Proxy" into the server name box and write "mc.theanarchistsproxy.com" into the server address box.
